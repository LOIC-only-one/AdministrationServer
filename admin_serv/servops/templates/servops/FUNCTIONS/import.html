Comment ça fonctionne ?

Vous pouvez importer des données depuis un fichier CSV. Les données doivent être formatées de la manière suivante :

  • Les colonnes doivent être séparées par des virgules
  • La première ligne doit contenir les noms des colonnes
  • Les applications et services doivent être séparés par des virgules

Voici un exemple de données formatées correctement :

Exemple pour ajotuer un service :

            name,launch_date,memory_used,required_memory,launch_server
            Service 1,2024-05-01,1024,2048,Serveur 1

Exemple pour ajouter une application :

            name,logo,user,services,launch_server
            Application 1,logos/logo1.png,John Doe,Service 1|Service 2,Serveur 1
