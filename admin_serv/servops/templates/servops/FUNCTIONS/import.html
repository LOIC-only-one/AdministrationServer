Comment ça fonctionne ?

Vous pouvez importer des données depuis un fichier CSV. Les données doivent être formatées de la manière suivante :

  • Les colonnes doivent être séparées par des virgules
  • La première ligne doit contenir les noms des colonnes
  • Les colonnes doivent être dans l'ordre suivant : Nom, Type de Serveur, Nombre de Processeurs, Capacité de Mémoire, Capacité de Stockage, Espace de Stockage Utilisé, Applications, Services
  • Les applications et services doivent être séparés par des virgules

Exemple de données

Voici un exemple de données formatées correctement :

            Nom,Type de Serveur,Nombre de Processeurs,Capacité de Mémoire,Capacité de Stockage,Espace de Stockage Utilisé,Applications,Services
            Serveur 1,Type 1,4,16,1000,500,Application 1,Service 1,Service 2
            Serveur 2,Type 2,8,32,2000,1000,Application 2,Application 3,Service 3
